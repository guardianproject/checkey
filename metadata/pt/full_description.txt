Checkey é um utilitário para obter informação sobre as APKs que estão instaladas no seu dispositivo. Começando com uma lista de todas as apps que estão instaladas no seu dispositivo, esta irá mostrar-lhe a assinatura APK com um único toque, e proporciona hiperligações para virustotal.com e androidobservatory.org para aceder facilmente aos perfis dessa APK. Também permite exportar o certificado de assinatura e gerar os ficheiros de código PIN ApkSignaturePin para usar com a biblioteca TrustedIntents.
